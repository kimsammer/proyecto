PROJECT X

Project X es una aplicación web que funciona como un organizador super completo que cuenta con funciones para llevar al día tus clases, eventos, tareas, exámenes, etc, haciendo que la organización y programación de estos sea efectiva y precisa.

funciones de Project X

  • Calcula todas tus notas y partiendo  de esos datos, te hace un análisis completo para saber que materias debes mejorar, cuáles te quedan o cuales tienen la mejor puntuación.
  • Tiene un sistema de programación y notificaciones para gestionar los tiempos de entrega de tus exámenes y tareas, avisando cuando estos tienen que ser entregados, o sí el examen que tienes mañana pertenece a una materia con bajo promedio.
  • la aplicación te permite guardar archivos multimedia en la nube
  • La aplicación hace un análisis completo de todas tus notas y actividades hechas o no, para ser representadas por los distintos gráficos con que cuenta la aplicación.
  • La aplicación crea un informe donde muestra todo tu rendimiento y las estadísticas con respecto a toda la información escolar dentro de la aplicación.
  • la aplicación cuenta con todo tipos de gráficas, desde estadísticas, hasta informes y calendarios personalizados
  • La aplicación cuenta con una interfaz modificable, donde podrás cambiar desde fuentes y colores hasta gráficos y estructuras.
  • La aplicación está hecha para el manejo y análisis de distintos datos que se almacenan en nuestros servidores, con el objetivo de llevar una organización completa de todo tu rendimiento escolar, donde la misma aplicación te brinda todo tipo de informes, gráficos, estadísticas, análisis inteligentes, recordatorios, ect.
